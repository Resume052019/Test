Linking Documents:

RevatureBlank
_blank load resource in new window. some browsers ;new tab depending on the user’s settings

Hyperlink Example_self RevatureSelf
_self tells the browser to load the resource in same window as current page. default value

Hyperlink Example_top RevatureTop
_top similar to parent but will find ancestor of all contexts or very first parent.

Hyperlink Example_parent RevatureParent
_parent load resource into parent of current context. Meaning if you’re currently viewing data in a frame, load resource in frame’s parent window. If no parent available; use current window/browsing context.
